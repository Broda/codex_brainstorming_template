# Idea Catalog

Canonical index of idea records and governance traceability.

## Catalog Rules

- Every idea ID appears once in this catalog.
- Status must match one of: `inbox`, `active`, `parked`, `killed`, `exported`.
- Strategic decisions must include ADR references.
- Exported ideas must include export file path.

## State Sources

- Inbox source: `ideas/_inbox.md`
- Active source: `ideas/_active.md`
- Parked source: `ideas/_parked.md`
- Killed source: `ideas/_killed.md`
